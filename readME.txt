Traveladvisor Project
Overview
  Traveladvisor is a simple travel planning website built with HTML, CSS, JavaScript, and Bootstrap. The project helps users explore destinations, search for cities, and view live weather information.
Features
  * Search for travel destinations
  * View live weather data using the Open Weather API
  * Explore popular cities
  * Filter destinations by region
  * Save favorite destinations
  * Store recent searches using LocalStorage
  * Responsive design with Bootstrap
Technologies used
  * HTML
  * CSS
  * JavaScript
  * Bootstrap
  * Open Weather API
  * LocalStorage
